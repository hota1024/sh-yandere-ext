# Yandere Web Extention

## 📁 フォルダ構成

```
┌ src
├── background - 拡張機能のバックグラウンドヤンデレ
├── common     - 全てのWebページに対応したヤンデレ
├── core       - API や 設定を管理するヤンデレ
├── instagram  - Instagram ヤンデレ(お休み中)
├── twitter    - Twitter ヤンデレ
├── ui         - 設定やログインができるヤンデレ
├── utils      - ヤンデレが使う便利な機能
└── youtube    - YouTube ヤンデレ
```
